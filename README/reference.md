## Contributing

When adding new documentation:

1. Place **Tutorials** in the `Tutorials/` directory
2. Place **Infrastructure guides** in the `How To Guides/infra_setup_guides/` directory
3. Place **Robot Framework guides** in the `How To Guides/robot_framework_guides/` directory
4. Place **Testing strategy guides** in the `How To Guides/testing_stretegy_guides/` directory
5. Update this reference document with links to new content
6. Follow the established naming convention: `snake_case.md` or `Title Case.md` as appropriate
